Atue como um Engenheiro de Software Sênior especializado em Python e MySQL 9.1. Gere o código inicial para o projeto 'ia-sdlc', um MVP de Plano Anual de Contratações (PCA) municipal.

Estrutura Técnica:

    models.py: Use SQLAlchemy para definir as tabelas: Usuario (id, nome, login, senha, secretaria), Secretaria (id, nome) e Contratacao (id, codigo, exercicio, objeto, descricao, valor_estimado, dotacao, data_planejada, secretaria_id como FK).

    app.py: Crie uma aplicação Flask que conecte ao MySQL no host 'mysql' (para o Docker no Debian) ou 'localhost' (para testes no Windows).

    Funcionalidades Necessárias:

        * Rota de Admin para cadastrar Usuarios, Secretarias e Contratações.
	* O campo codigo das contratações deverar ser gerado automaticamente com a seguinte estrutura:
	 - PCA  
	 - ITEM  
	 - ANO  
	 - ID da secretaria
	 Exemplo: PCA-100.2026-5
	 Onde 100 é o id da contratação, 2026 é o ano do exercicio e 5 o id da secretaria 

        Rota de Visualização de Usuário com filtros por Secretaria e exercicio.

        Lógica de exportação dos dados filtrados para PDF (usando ReportLab) e Excel (usando Pandas/OpenPyXL).

    Foco em Qualidade (SDLC): Escreva um código limpo, mas inclua comentários explicativos onde a IA tomou decisões de arquitetura. O código deve estar pronto para ser analisado pelo SonarQube em busca de code smells e vulnerabilidades.
